# ito ゲーム 数字配布ツール（Spring Boot）

## 必要な環境
- Java 17 以上
- Maven 3.6 以上

## 起動方法

```bash
cd ito-spring
mvn spring-boot:run
```

起動後、以下のURLにアクセス：

| 役割 | URL |
|------|-----|
| 参加者（司会含む全員） | http://localhost:8080/ |
| 進行役管理画面 | http://localhost:8080/admin |

## 使い方

### 進行役
1. `/admin` を開く
2. お題を選択または入力して「ラウンド開始」をクリック
3. 参加者URLをTeamsのチャットに貼り付けて共有
4. 全員が数字を引いたらゲームスタート
5. ラウンド終了後は「ラウンド終了」→次のラウンドは「ラウンド開始」で新しいお題を設定

### 参加者（司会も同様）
1. 共有されたURLをブラウザで開く
2. ラウンドが始まったら自分の名前を入力
3. 「数字を引く」ボタンを押すと自分だけに数字が表示される
4. その数字をお題に沿った言葉で表現してゲーム開始！

## 注意事項
- サーバーを再起動するとゲームデータはリセットされます
- 同一LAN内での使用を想定しています（社内イベント向け）
- 同じ名前で引いた場合は同じ数字が返ります（リロード対応）
